참고:  Microsoft Corporation(또는 거주 지역에 따라 계열사 중 하나)은 귀하에게 본 추가 구성 요소의 사용권을 허여합니다. 추가 구성 요소는 Microsoft Dynamics 365 온라인 서비스의 각 유효하게 라이선스가 허여된 정기가입을 통해 사용할 수 있습니다. 해당 온라인 서비스 사용권이 없는 경우에는 본 추가 구성 요소도 사용할 수 없습니다. 본 추가 구성 요소 사용 시 해당 온라인 서비스 사용권 계약이 적용됩니다. 적용되는 사용권 계약 또는 (http://go.microsoft.com/fwlink/?LinkId=512132에서 확인할 수 있는) Microsoft 온라인 서비스 개인정보취급방침의 상충되는 내용에도 불구하고, 추가 구성 요소의 사용 시 다음의 추가 조건이 적용됩니다.

이 추가 구성 요소는 귀하가 독점적으로 사용하기 위해 전용 데이터센터에 프로비전되지 않으며 온라인 서비스의 일부나 온라인 서비스의 구성요소도 아닙니다. 본 추가 구성 요소의 사용은 Dynamics 365 Online for Government에 적용되는 제품별 조항 또는 제품별 조항이 등록계약 또는 계약(수정계약, 사용권 또는 제품 조건 포함)에 포함될 수 있는 다른 온라인 서비스에 적용되지 않습니다.  온라인 서비스 약관 또는 Microsoft 개인정보취급방침에 제공된 내용과 상반되는 조항에도 불구하고 이 추가 구성 요소는 온라인 서비스 약관의 데이터 처리 조항에 따라 제공되지 않습니다. 본 추가 구성에는 Microsoft Trust Center에서 작성한 참조나 진술이 적용되지 않습니다.

이 추가 구성 요소는 “있는 그대로”, “모든 오류가 포함된 상태로” 및 “사용할 수 있는 상태로” 제공되며 온라인 서비스 조건 및 이곳에서 참조하는 자료에 명시된 판매 및 제한적 보증에서 제외됩니다. 이 추가 구성 요소에는 고객 지원이 적용되지 않습니다.

본 추가 조건에 동의하지 않으면, 이 추가 구성 요소를 설치 또는 사용하지 마십시오.
